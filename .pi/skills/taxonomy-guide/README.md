# Taxonomy Guide

Project-local skill that defines the naming taxonomy for
skills and extensions in this package.

Covers domains, type suffixes, location rules (package-bound
vs project-local) and the decision framework for when to
create a new skill, a new extension or add guidance to
AGENTS.md.

This skill is in `.pi/skills/` because it's about developing
this package, not about general methodology that consumers
would use.
