# quest-format

Skill: how a quest README and its documents are structured.

This skill is loaded by pi when it matches the description in
[SKILL.md](./SKILL.md). It teaches the on-disk format: front
matter, the four core and four optional sections, emoji
glyphs, IDs, Cast bullets, Journey entries.

Read [SKILL.md](./SKILL.md) for the full content. Pair with
[quest-convention](../quest-convention/SKILL.md) for the
choices around kind, promotion, reordering and scaffolding.
