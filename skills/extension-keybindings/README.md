# Extension Keybindings Skill

Teaches the keybinding system that all Pi extensions follow.
Loaded when writing or modifying extensions that have
user-facing panels, gates or interactive prompts.

Covers the five rules (no global shortcuts, numbers for views,
Enter/Escape for binary gates, explicit letters for
destructive actions, Shift = annotate), the universal input
model, the six-verb set, the footer grid layout and guidance
for adding new gates and workspaces.
