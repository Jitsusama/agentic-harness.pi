# subagent-fleet-guide

Methodology for the `subagent` tool. Teaches when to fan
N pi subagents out concurrently, how to shape personas
through system prompts, and how to keep cost and
cancellation honest.

Loaded automatically when the user asks for "various
angles", "multiple perspectives", "parallel
investigation" or anything else the description matches.

Pairs with the `subagent-workflow` extension (the
substrate) and `prose-standard` (voice rules for the
synthesised output the host agent writes back to the
user).

See `SKILL.md` for the full guidance.
