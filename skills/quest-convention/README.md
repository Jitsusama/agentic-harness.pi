# quest-convention

Skill: operational conventions for the quest system.

This skill is loaded by pi when it matches the description in
[SKILL.md](./SKILL.md). It teaches when to use which kind,
how to reorder priorities, when to scaffold documents, how
to conclude versus retire, the resuscitate pattern.

Read [SKILL.md](./SKILL.md) for the full content. Pair with
[quest-format](../quest-format/SKILL.md) for the on-disk
shape.
