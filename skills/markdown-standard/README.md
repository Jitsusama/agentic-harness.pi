# Markdown Standard

Teaches the agent markdown structure: Title Case headings,
the 80-character wrap target, reference-style links, fenced
code blocks, tables and lists.

Activates when writing or editing any markdown file, or when
adding a heading, link, table or code block.

Owns markdown structure. Pairs with
[prose-standard](../prose-standard/), which owns voice,
grammar, spelling and punctuation.
