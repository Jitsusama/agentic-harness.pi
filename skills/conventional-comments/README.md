# Conventional Comments Skill

Teaches the agent the [Conventional Comments](https://conventionalcomments.org/)
format for review feedback — labels, decorations, and tone guidance.

Activates when writing review comments, giving code feedback, or
formatting review observations. Reusable across pr-review, pr-reply,
pr-annotate, and casual code feedback.

Independent of any specific review workflow — can be loaded standalone
whenever the agent needs to format feedback.
